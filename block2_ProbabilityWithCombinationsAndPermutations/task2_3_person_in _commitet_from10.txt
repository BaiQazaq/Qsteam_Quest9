Количество способов сформировать комитет из 3 человек из группы из 10 человек может быть вычислено с использованием сочетаний. 
Сочетание C(n,k) означает количество способов выбрать 
k элементов из множества из 
n элементов без учета порядка.

        C(n,k)= n! / k!(n−k)!

В данном случае, мы хотим выбрать комитет из 3 человек из группы из 10 человек. Поэтому:

        C(10,3) = 10! / (3!(10−3)!)
                = 10! / (3! * 7!)
                = 3 628 800 / (6 * 5 040)
                = 3 628 800 / 30 240
                = 120

Таким образом, есть 120 различных способов сформировать комитет из 3 человек из группы из 10 человек.
