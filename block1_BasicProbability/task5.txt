Для вычисления вероятности не вытянуть карту с лицевым значением (валет, дама, король) из колоды карт, 
нужно разделить количество благоприятных исходов (не лицевые карты) на общее количество возможных исходов.

В стандартной колоде 52 карты, каждая масть имеет по одному валету, даме и королю (всего 3 лицевые карты в каждой масти).

Количество не лицевых карт в каждой масти:

    Количество не лицевых карт=13−3=10

Таким образом, количество благоприятных исходов для одной масти равно 10.

Вероятность (P) может быть вычислена по формуле:

    P(Не лицевая карта) = Количество благоприятных исходов / Общее количество возможных исходов

Подставим значения:
        P(Не лицевая карта)= 10 / 13

Таким образом, вероятность не вытянуть карту с лицевым значением из стандартной колоды карт составляет 
            10 / 13​
            или 
            примерно 0.7692 (76.92%).
