# 4. Calculate the probability of drawing an Ace from a standard deck of playing cards.

# Для вычисления вероятности вытянуть туз из стандартной колоды игральных карт, 
# нужно разделить количество благоприятных исходов (вытягивание туза) на общее количество возможных исходов.

# В стандартной колоде 52 карты, и всего есть 4 туза (по одному в каждой масти).

# Вероятность (P) может быть вычислена по формуле:
#               P(Туз)= Общее количество возможных исходов / Количество благоприятных исходов 

# Подставим значения:
#               P(Туз)= 4 / 52

# Таким образом, вероятность вытянуть туз из стандартной колоды игральных карт составляет 
#               4 / 52 или примерно 0.0769 (7.69%).
